# homebrew-installguard

Homebrew tap for [InstallGuard](https://github.com/jt-systems/installguard) — a policy gate for npm/pnpm/yarn installs.

## Install

```bash
brew tap jt-systems/installguard
brew install installguard
```

## Updates

This tap is updated automatically by the [InstallGuard release workflow](https://github.com/jt-systems/installguard/blob/main/.github/workflows/release.yml) on each tagged version. The current formula lives in [`Formula/installguard.rb`](Formula/installguard.rb).
